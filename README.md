# Contact Manager Python

Projet réalisé dans le cadre de mon apprentissage du développement Python par projet.

## Description

Application en ligne de commande permettant de gérer un carnet d'adresses.

Les données sont sauvegardées automatiquement dans un fichier JSON afin d'être conservées entre plusieurs exécutions du programme.

## Fonctionnalités

- Ajouter un contact
- Afficher les contacts
- Rechercher un contact
- Modifier un contact
- Supprimer un contact
- Validation du numéro de téléphone
- Sauvegarde JSON
- Chargement automatique des données

## Technologies utilisées

- Python
- JSON
- os
